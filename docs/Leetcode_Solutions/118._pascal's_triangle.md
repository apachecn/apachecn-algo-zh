### 118. Pascal's Triangle

题目:

<https://leetcode.com/problems/pascals-triangle/>


难度:

Easy


高中数学知识，把行数理理清楚就ok


```python
class Solution(object):
    def generate(self, numRows):
        """
        :type numRows: int
        :rtype: List[List[int]]
        """
        if numRows == 0:
            return []
        res = [[1]]
        for i in range(1, numRows):
            tmp = [1]
            for j in range(1, i):
                tmp.append(res[-1][j-1]+res[-1][j])
            tmp.append(1)
            res.append(tmp)
        return res         
```
